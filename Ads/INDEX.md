- 파일명: `MM-DD 결정 요약.md`
- 소재 생성 로그 파일명: `YYYY-MM-DD asset-slug.md`
- 필수 섹션: 결정 / 근거 (숫자 테이블) / 기대 효과 / 재검토 조건
- 소재 생성 로그 필수: source / user request raw typed string / result R2 URL / manifest URL / prompt / learned tags
- 다음 작업은 `Ads/Creative Logs/INDEX` shortcut 먼저 확인, raw/prompt는 필요할 때만 개별 로그 확인
