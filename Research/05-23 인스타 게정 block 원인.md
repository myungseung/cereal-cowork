## 1. 왜 어떤 AI 계정은 살아남는가

핵심 = 기만 여부. 차단 트리거는 "AI라서"가 아니라 "진짜 인간인 척 속여서".

| 생존형                | 제거형                 |
| ------------------ | ------------------- |
| "AI 가상인플루언서" 투명 표시 | AI 생성물 표시 없음        |
| 예술/패션/라이프스타일/운동    | 노골적 유혹/성적 컨셉        |
| 플랫폼 정책 내 운영        | OnlyFans 등 유료사이트 유입 |
| 계정 1~3개 집중 관리      | 10개씩 mass 생성        |

생존 사례: Lil Miquela (290만 팔로워), Imma (일본), Rui (중국) — 전부 AI 공개 + 콘텐츠 IP 인정

## 2. 계정 삭제 직접 원인

"릴스 > 스토리 > 비즈니스 링크" + "가짜계정" 차단 사유

- 외부 링크(유료 전환) + AI 미표시 + 가짜계정 의심 = 메타 조합 적발 패턴
- 운동 릴스 100만뷰 = 유기 도달은 컸으나, 신규/저신뢰 계정이라 조회수 폭발 직후 신고 or 시스템 검토로 차단

| 항목      | 내용                                                             |

## 4. Reddit/Threads 사례

Reddit = AI 계정에 가장 가혹한 플랫폼
- 봇 자동 포스팅 = 몇 시간 내 shadowban
- datacenter IP + Playwright 자동화 = 탐지 확정
- 생존법: AI 초안 작성, 사람이 직접 수동 포스팅. Residential IP 필수. 계정당 30일 이상 워밍업

Threads = 상대적 덜 단속, but 메타 소속이라 정책은 인스타와 동일

## 결론

장기 전략 = AI 표시 + 주제 전환(운동/예술) + 1계정 집중 관리. "오늘 10개 더 생성"은 단기 조회수 가능하나 IP/디바이스 영구밴 누적 위험, 시간 지날수록 탐지 난이도 상승.
