# 일본어 버전 handoff

## 목표
- `/ja`로 들어온 유저에게 기존 Cereal을 일본어로 보여준다.

## 작동 구조
- URL/domain으로 일본어 유저를 판단한다.
- 화면 문구는 i18n 사전에서 일본어를 꺼내 쓴다.
- 기본 캐릭터는 DB를 바꾸지 않고 API 응답에서 일본어로 바꿔 내려준다.
- 채팅은 한국어/일본어 대화 기록을 따로 저장한다.
- `/ja` 채팅에서는 LLM에 일본어로 답하라고 지시한다.

## 변경된 범위
- Home 문구
- Create 문구
- Chat 문구
- Payment popup 문구
- 기본 캐릭터 이름/설명/first message
- 일본어 랜덤 이름

## 변경하지 않은 범위
- 화면 레이아웃
- 버튼 위치
- 결제 앱 이동 방식
- 유저가 직접 만든 캐릭터 이름/설명

- 일본어 카피 자연스러움 확인
