## 발견 2026-06-17: Grok + 예제 라이브러리 파이프라인

### 판정

- Grok 4.20 + well-working prompt library 조합 = context image 품질 확보
- adult/sex scene도 필터 없이 생성 가능 (Gemini Flash 대비 확실한 우위)
- 비용: $0.0007/call (Grok 4.20), 생성시간 3-8s

### 핵심

| 항목 | 값 |
| --- | --- |
| 문제 | agent가 작법 몰라도 패턴 기반으로 맥락+미감 유지 |
| 해결 | 예제 라이브러리 + Grok + z-image-turbo 조합 |
| 모델 | Grok 4.20 (prompt 생성) → z-image-turbo-lora (이미지 생성) |
| LoRA | NiceGirls 0.3 + Hyper-SD 0.2 (Mystic 불필요) |
| 증거 | `docs/06-17 grok sex scene prompt case.md` |

### 새 파이프라인

```
채팅 맥락 + 캐릭터 정보 → Grok 4.20 → 예제 라이브러리에서 유사 3개 선택
→ 새 scene prompt 생성 → z-image-turbo → 이미지
```

- Grok이 예제 라이브러리에서 가장 유사한 prompt 3개 retrieve
- 캐릭터 prompt + 채팅 맥락 반영하여 새로운 scene prompt 구성
- z-image-turbo로 생성 → R2 업로드
- agent가 작법 몰라도 예제 기반으로 퀄리티 보장
